⚠️ Critical Operational Notice ⚠️

All IP addresses, hostnames, and resource identifiers shown in this guide are illustrative only. Always use the values returned by your own CLI commands or automation outputs. Never copy example values directly into a live environment.
Cloud resources incur costs while provisioned, even when idle. Ensure all resources are cleaned up when you are finished:
Manual provisioning: Stop or terminate resources using your cloud provider's console or CLI.
Infrastructure as Code: Destroy all provisioned resources using the same tooling that created them (for example, terraform destroy).
Do not leave infrastructure running unintentionally!

Crucial Advice for this Structure
Since you are creating multiple files inside references/, you must add a README.md file inside that folder too.

If you don't have a README.md inside references/, when a user clicks that folder on GitHub or GitLab, they just see a raw file list. A README.md acts as the "Table of Contents" for that folder.
